# Easy-blog
这是一个本地可运行的轻量博客，包含前台展示与后台管理，数据存储在 SQLite 中，支持多用户创建。This is a lightweight blog that can run locally. It includes both a frontend display and a backend management system. The data is stored in SQLite and supports multiple users creating content.これはローカルで実行できる軽量なブログです。フロントエンド表示とバックエンド管理機能を備えており、データはSQLiteに保存され、複数ユーザーによる作成をサポートしています。
